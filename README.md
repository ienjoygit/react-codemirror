react-codemirror
===

CodeMirror component for React.

## Install

```bash
npm install @uiw/react-codemirror --save
```

## usage

```jsx
import CodeMirror from '@uiw/react-codemirror';

<CodeMirror
  value={code}
  options={{
    keyMap: 'sublime',
    mode: 'jsx',
  }}
/>
```
